# Garmin Connect Backup

`garmin-connect-backup` as a simple command line tool that downloads all available data from a Garmin Connect account.

## Installation

```sh
npm i -G garmin-connect-backup
garmin-connect-backup --configure
```

## Usage

```sh
garmin-connect-backup
```
